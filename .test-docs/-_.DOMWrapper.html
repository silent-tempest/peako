This class representing a wrapper for the DOM elements.

Parameters

selector (Node) - This element will be added to the new object.

selector (string) - When the selector string contains HTML content, it will be parsed and added to the new object, if it doesn't, then will be added elements selected by the CSS selector.

selector (ArrayLikeObject) - Elements from selector will be copied to the new object.

selector (function) - It's the shorthand for _( document ).ready().

Example

Handle the page loaded event

var onready = function () {
  _( '#element' ).text( 'The page is loaded' );
};

// Two lines below will do the same things.
_( document ).ready( onready );
_( onready );

get( [index] ) -> {Node|Array}

peako.js - line 3857

Returns the element at the index position (may be negative for counting from the end). If index isn't passed, then the built-in array with all elements of this object will be returned.

Parameters

[index] (number) - The position of the needed element.

Returns

(Node|Array) - Returns the picked element or array of all elements.

Examples

Pick a specific element

var $div = _( 'div' ),
    $firstDiv = $div.get( 0 ),
    $lastDiv = $div.get( -1 );

Get all elements

// Get all elements and use bulit-in array method.
var selected = _( 'div' ).get().find( function ( div ) {
  return /stuff-\d+-selected/.test( div.className );
} );
addClass( classes )

Add classes to all elements.

Parameters

classes (string|function) - The classes that should be added to the elements, or the function that will be called (with the element index and current element classes) once for each element, and returns the classes that need to be added.

Examples

Using a string

_( 'div' ).addClass( 'something blue' );
_( document.body ).addClass( 'mobile' );

Using a function

_( 'tr' ).addClass( function ( rowIndex, className ) {
  if ( rowIndex % 2 ) {
    return 'light';
  }

  return 'light-grey';
} );
